# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/zhengliyoux/betterfly.git
cd betterfly
```

## Install Dependencies

### npm

```bash
npm install
```

### yarn

```bash
yarn install
```

### pnpm

```bash
pnpm install
```

---

# ▶️ Running Project

```bash
npm run dev
```

Atau:

```bash
yarn dev
```
