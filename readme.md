# 📦 Installation

Clone repository:

```bash
git clone https://github.com/zhengliyoux/betterfly.git
cd betterfly
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run Project

```bash
npm start
```

atau:

```bash
node index.js
```

---

# 🌐 Deployment

Project ini support deployment menggunakan:

- Vercel
- Node.js
